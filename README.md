### 🩺 QuickAid – Emergency Health Support Platform (Hackathon Prototype)

**QuickAid** is a web-based emergency health support prototype built for a hackathon challenge. It connects users in critical situations with the nearest available hospitals, emergency responders, and provides quick first-aid guidance based on the type of emergency.

🔴 **Problem:** Many Nigerians face delays during medical emergencies due to lack of access to accurate hospital information, ambulance response, or public assistance.

✅ **Solution:** QuickAid helps users:

- Select the type of emergency (e.g. gunshot, asthma, bleeding)
- Get matched with the nearest suitable hospitals
- Automatically alert hospital ambulance services
- Provide static first-aid guidance during critical moments
- Store basic medical information securely for responders

🎯 **Goal:** Save time. Save lives. Digitize Nigeria’s emergency response system through tech.

---

### 🚀 Features (MVP)

- Emergency type classification and smart hospital filtering
- Ambulance alert system (simulated)
- First aid instructions for different emergencies
- Secure user profile (e.g., blood type, allergies, pre-existing conditions)
